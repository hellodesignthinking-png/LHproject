## 1.5 ROI 시뮬레이션 예시 (서울 800㎡ 기준)⁵

### 기본 시나리오

```
[토지 정보]
위치: 서울특별시 마포구 월드컵북로 120
면적: 800㎡ (242평)
용도지역: 제2종일반주거지역
건폐율: 60%, 용적률: 200%

[사업비 산정]
1. 토지비
   - 감정평가액: ₩5,000,000/㎡
   - 총 토지비: ₩4,000,000,000 (40억 원)

2. 건축비
   - 건축면적: 480㎡ (건폐율 60%)
   - 연면적: 1,600㎡ (용적률 200%)
   - 단위 공사비: ₩2,000,000/㎡ (서울 기준)
   - 총 건축비: ₩3,200,000,000 (32억 원)

3. 부대비용 (사업비의 8%)
   - 설계비: ₩200,000,000 (2억 원)
   - 인허가비: ₩150,000,000 (1.5억 원)
   - 기타 부담금: ₩226,000,000 (2.26억 원)
   - 소계: ₩576,000,000 (5.76억 원)

4. 금융비용 (대출 70%, 금리 5%, 2년)
   - 대출액: (40억 + 32억 + 5.76억) × 0.7 = ₩5,443,200,000
   - 이자: ₩544,320,000 (5.44억 원)

[총 사업비]
₩4,000M + ₩3,200M + ₩576M + ₩544M = ₩8,320M (83.2억 원)

[LH 매입가] (공사비 연동형, 인정이익 4%)
매입가 = 토지비 + 표준건축비 + 인정이익
       = ₩4,000M + ₩3,200M + (₩3,200M × 0.04)
       = ₩4,000M + ₩3,200M + ₩128M
       = ₩7,328M (73.28억 원)

[손익 분석]
매출(매입가): ₩7,328M
원가(사업비): ₩8,320M
손익: -₩992M (-9.92억 원) ← 적자 발생!

[문제점 분석]
→ 금융비용이 과도 (5.44억, 사업비의 6.5%)
→ 대출비율을 50%로 낮추거나
→ 자기자본 비중 증가 필요
```

## 4.5 사업성 평가 상세 (80점 만점)

### 4.5.1 ROI 및 수익성 (30점)

**ROI 계산식**:
```
ROI = (LH 매입가 - 총 사업비) / 총 사업비 × 100

총 사업비 = 토지비 + 건축비 + 부대비용 + 금융비용
LH 매입가 = MAX(감정가, 공사비 연동가)
```

## 4.9 자동 탈락(Rejection) 조건

다음 조건 중 하나라도 해당 시 **자동 탈락** 처리됩니다:

| No. | 탈락 조건 | ZeroSite 검증 |
|-----|-----------|---------------|
| 1 | 대지면적 500㎡ 미만 | `land_area < 500` |
| 2 | 건축 세대수 10세대 미만 | `units < 10` |
| 3 | 건폐율 초과 | `actual_coverage > legal_coverage` |
| 4 | 용적률 초과 | `actual_far > legal_far` |
| 5 | 일조권 미확보 (연속 2시간 미만) | `sunlight_hours < 2` |
| 6 | 맹지 또는 접도 조건 미충족 | `road_width < 4 or road_length == 0` |
| 7 | 녹지지역·개발제한구역 | `zone_type in ['녹지지역', '개발제한구역']` |
| 8 | ROI 4% 미만 | `roi < 4.0` |
| 9 | 토지 소유권 불명확 | `ownership_status != 'confirmed'` |
| 10 | LH 임대료 기준 초과 (시세 80% 이상) | `rent_ratio > 0.8` |

**ZeroSite 구현**:
```python
def check_auto_rejection(analysis_result):
    rejections = []
    if analysis_result['land_area'] < 500:
        rejections.append("대지면적 500㎡ 미만")
    if analysis_result['building_capacity']['units'] < 10:
        rejections.append("건축 세대수 10세대 미만")
    # ... (생략)
    return rejections
```

## 5.1 개요

본 장에서는 LH 신축매입임대 사업에서 **즉시 탈락(Rejection)**, **조건부 보류(Hold)**, **제외 대상(Exclusion)**으로 분류되는 30개 이상의 세부 기준을 제시하고, 각 항목에 대한 ZeroSite 자동진단 매핑을 명시합니다.

## 5.2 탈락 기준 분류 체계

| 분류 | 코드 | 의미 | ZeroSite 처리 |
|------|------|------|---------------|
| **즉시 탈락** | R (Reject) | 심사 진행 불가, 자동 탈락 | `status: "REJECT"` |

## 6.2 탄소배출량 산정

### 6.2.1 건축물 탄소배출량 산식

**건축물 생애주기 탄소배출량 (LCA: Life Cycle Assessment)**

```
총 탄소배출량 (kgCO₂eq) = 자재생산 + 운송 + 시공 + 운영 + 해체

자재생산 배출량 = Σ(자재 중량i × 탄소배출계수i)
운영 배출량 (연간) = 에너지사용량 (kWh/㎡·년) × 전력 탄소배출계수
```

**ZeroSite 적용 산식 (간소화)**:
```python
def calculate_carbon_emission(building_area, energy_efficiency_grade):
    # 기준 연간 에너지 사용량 (kWh/㎡·년)
    base_energy = {
        '1등급': 60,
        '2등급': 90,
        '3등급': 120,
        '4등급': 150,
        '5등급': 180
    }
    
    # 전력 탄소배출계수 (2024년 기준: 0.4533 kgCO₂eq/kWh)
    carbon_factor = 0.4533
    
    # 연간 탄소배출량
    annual_emission = building_area * base_energy[energy_efficiency_grade] * carbon_factor
    
    # 30년 생애주기 탄소배출량
    lifecycle_emission = annual_emission * 30
    
    return {
        'annual_emission_kg': annual_emission,
        'lifecycle_emission_tons': lifecycle_emission / 1000
    }
```

**예시 계산 (서울 800㎡, 에너지효율 1등급)**:
```
건축 연면적: 1,600㎡ (용적률 200%)
연간 에너지 사용량: 60 kWh/㎡·년
연간 탄소배출량: 1,600 × 60 × 0.4533 = 43,516 kgCO₂eq (43.5톤)
30년 생애주기: 43.5 × 30 = 1,305톤 CO₂eq
```

### 6.3.2 에너지 절감률 계산

```
에너지 절감률 (%) = (기준 에너지 - 실제 에너지) / 기준 에너지 × 100

기준 에너지 = 「건축물의 에너지절약설계기준」에 따른 의무 기준
실제 에너지 = 에너지 성능 시뮬레이션 결과 (ECO2, EnergyPlus 등)
```

**ZeroSite 구현 (추정)**:
```python
def calculate_energy_savings(zone_type, actual_energy):
    # 기준 에너지 (의무 기준)
    baseline_energy = {
        '제1종일반주거지역': 180,
        '제2종일반주거지역': 180,
        '제3종일반주거지역': 180,
        '준주거지역': 180
    }
    
    baseline = baseline_energy.get(zone_type, 180)
    savings_rate = (baseline - actual_energy) / baseline * 100
    
    return max(0, savings_rate)
```

### 6.4.2 태양광 발전 설치 면적 산정

**필요 설치 면적 계산**:
```
태양광 설치 면적 (㎡) = 목표 발전량 (kWh/년) / (태양광 패널 효율 × 일사량 × 시스템 효율)

태양광 패널 효율: 0.20 (20%, 일반적인 다결정 실리콘)
일사량: 서울 기준 4.0 kWh/㎡·day = 1,460 kWh/㎡·년
시스템 효율: 0.80 (인버터 손실, 온도 손실 등)
```

**예시 계산 (서울 1,600㎡ 건물, 에너지효율 1등급)**:
```
연간 에너지 사용량: 1,600 × 60 = 96,000 kWh/년
신재생 목표 (15%): 96,000 × 0.15 = 14,400 kWh/년

태양광 설치 면적 = 14,400 / (0.20 × 1,460 × 0.80)
                 = 14,400 / 233.6
                 = 61.6 ㎡

옥상 면적 (800㎡) 대비: 61.6 / 800 = 7.7%
```

**ZeroSite 구현**:
```python
def calculate_solar_panel_area(total_energy, renewable_ratio=0.15):
    target_generation = total_energy * renewable_ratio
    
    panel_efficiency = 0.20
    solar_irradiation = 1460  # kWh/㎡·년 (서울 기준)
    system_efficiency = 0.80
    
    required_area = target_generation / (panel_efficiency * solar_irradiation * system_efficiency)
    
    return {
        'target_generation_kwh': target_generation,
        'required_area_sqm': round(required_area, 1),
        'panel_count': int(required_area / 1.7)  # 1.7㎡/패널 가정
    }
```

```python
# ROI 계산
```
