## 1.3 2025년 LH 정책 변화 핵심 요약³

### 5대 핵심 변화 사항

#### (1) 공사비 연동형 주택 전면 확대
```
[변경 내역]
기존: 청년형, 신혼I형만 적용
2025: 전 7개 유형 적용 (일반형, 고령자형 포함)

[매입가 산정 방식]
매입가 = 토지비 + 표준건축비 + 인정이익

표준건축비 = min(실제 공사비, 지역별 상한액)
인정이익 = 표준건축비 × 3~5% (지역별 차등)

[ZeroSite 대응]
BuildingCalculator 모듈에 지역별 상한액 DB 구축
ROI Engine에서 공사비 시나리오 3개 제공 (보수/기준/낙관)
```

#### (2) 착공 중심 심사 체계 강화
```
[신규 프로세스]
심사 → 협약 → [착공 승인 심사] → 공사 → 준공

[착공 승인 심사 항목]
✓ 재무 건전성: 부채비율 200% 이하, 유동비율 150% 이상
✓ 신용등급: BB+ 이상 (건설사 기준)
✓ 자금 조달 확정: 총 사업비의 30% 자기자본 확인
✓ 설계 완성도: 80% 이상 (구조도, 설비도 포함)

[ZeroSite 대응]
LHCriteriaChecker에 착공심사 체크리스트 자동 생성
재무지표 자동 계산 및 적정성 판단
```

#### (3) ESG 평가 비중 증가 (신설 45점)
```
[ESG 평가 배점표]
환경(E): 30점
- 제로에너지 건축물 인증: 10점
- 신재생에너지 20% 이상: 10점
- 친환경 자재 30% 이상: 10점

사회(S): 30점
- 취약계층 공급 30%: 15점
- 지역사회 기여: 15점

지배구조(G): 15점
- 내부통제 시스템: 7점
- 투명한 하도급 관리: 8점

[가점 효과]
30점 이상: 입찰 우선권
40점 이상: 금리 우대 0.2%p
45점: 사업비 3% 추가 지원

[ZeroSite 대응]
ESG Auto-Calculator (v5.1 구현 예정)
탄소배출량 자동 환산
에너지효율 등급 자동 평가
```

#### (4) 수요 예측 모델 고도화
```
[기존 vs 신규 비교]
기존: 통합 수요 점수 1개
신규: 유형별 독립 점수 7개 (청년, 신혼I/II, 다자녀, 고령자, 일반, 든든전세)

[수요 점수 산정 공식]
총점 = (입지 × 0.30) + (규모 × 0.25) + (사업성 × 0.30) + (법규 × 0.15)

유형별 가중치 차등 적용:
- 청년형: 지하철 접근성 가중치 1.5배
- 신혼형: 초등학교 접근성 가중치 1.5배
- 고령자형: 병원 접근성 가중치 2.0배

[ZeroSite 구현]
_calculate_type_demand_scores() 메서드 (598 lines)
실시간 인구통계 API 연동 (MOIS API)
```

## 1.5 ROI 시뮬레이션 예시 (서울 800㎡ 기준)⁵

### 기본 시나리오

```
[토지 정보]
위치: 서울특별시 마포구 월드컵북로 120
면적: 800㎡ (242평)
용도지역: 제2종일반주거지역
건폐율: 60%, 용적률: 200%

[사업비 산정]
1. 토지비
   - 감정평가액: ₩5,000,000/㎡
   - 총 토지비: ₩4,000,000,000 (40억 원)

2. 건축비
   - 건축면적: 480㎡ (건폐율 60%)
   - 연면적: 1,600㎡ (용적률 200%)
   - 단위 공사비: ₩2,000,000/㎡ (서울 기준)
   - 총 건축비: ₩3,200,000,000 (32억 원)

3. 부대비용 (사업비의 8%)
   - 설계비: ₩200,000,000 (2억 원)
   - 인허가비: ₩150,000,000 (1.5억 원)
   - 기타 부담금: ₩226,000,000 (2.26억 원)
   - 소계: ₩576,000,000 (5.76억 원)

4. 금융비용 (대출 70%, 금리 5%, 2년)
   - 대출액: (40억 + 32억 + 5.76억) × 0.7 = ₩5,443,200,000
   - 이자: ₩544,320,000 (5.44억 원)

[총 사업비]
₩4,000M + ₩3,200M + ₩576M + ₩544M = ₩8,320M (83.2억 원)

[LH 매입가] (공사비 연동형, 인정이익 4%)
매입가 = 토지비 + 표준건축비 + 인정이익
       = ₩4,000M + ₩3,200M + (₩3,200M × 0.04)
       = ₩4,000M + ₩3,200M + ₩128M
       = ₩7,328M (73.28억 원)

[손익 분석]
매출(매입가): ₩7,328M
원가(사업비): ₩8,320M
손익: -₩992M (-9.92억 원) ← 적자 발생!

[문제점 분석]
→ 금융비용이 과도 (5.44억, 사업비의 6.5%)
→ 대출비율을 50%로 낮추거나
→ 자기자본 비중 증가 필요
```

### 민감도 분석 (±10%)

| 변수 | -10% | 기준 | +10% | 영향도 |
|------|------|------|------|--------|
| **토지비** | 적자 6.0억 | 적자 9.9억 | 적자 13.9억 | **매우 높음** |
| **건축비** | 적자 6.7억 | 적자 9.9억 | 적자 13.1억 | **매우 높음** |
| **금리** | 적자 9.4억 | 적자 9.9억 | 적자 10.5억 | 보통 |
| **인정이익률** | 적자 10.2억 | 적자 9.9억 | 적자 9.6억 | 낮음 |

### 수익 개선 시나리오

```
[시나리오 A: 자기자본 50% 투입]
대출액: ₩4,160M (50%)
이자: ₩416M (4.16억 원, 2년)
총 사업비: ₩8,192M
손익: ₩7,328M - ₩8,192M = -₩864M (여전히 적자)

[시나리오 B: 공사비 10% 절감 + 인정이익 5%]
건축비: ₩2,880M (32억 × 0.9)
인정이익: ₩144M (₩2,880M × 0.05)
매입가: ₩4,000M + ₩2,880M + ₩144M = ₩7,024M
총 사업비: ₩7,960M (금융비용 포함)
손익: ₩7,024M - ₩7,960M = -₩936M (여전히 적자)

[시나리오 C: 비수도권 사업 (경기 외곽)]
토지비: ₩2,000M (50% 절감)
건축비: ₩2,400M (25% 절감)
매입가: ₩2,000M + ₩2,400M + (₩2,400M × 0.04) = ₩4,496M
총 사업비: ₩5,200M
손익: ₩4,496M - ₩5,200M = -₩704M (여전히 적자)

[결론]
→ 서울 도심 사업은 수익성 확보 어려움
→ 경기 외곽 또는 지방 광역시 사업 권장
→ 자기자본 비중 50% 이상 필요
→ LH 인정이익률 5% 확보가 핵심
```
