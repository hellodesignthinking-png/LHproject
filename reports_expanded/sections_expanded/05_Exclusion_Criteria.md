# 제5장. 상세 제외·탈락·보류 기준 (30+ 항목)

---
**워터마크**: ZeroSite | ZeroSite Land Report v5.0  
**페이지**: 19-26

---

## 5.1 개요

본 장에서는 LH 신축매입임대 사업에서 **즉시 탈락(Rejection)**, **조건부 보류(Hold)**, **제외 대상(Exclusion)**으로 분류되는 30개 이상의 세부 기준을 제시하고, 각 항목에 대한 ZeroSite 자동진단 매핑을 명시합니다.

---

## 5.2 탈락 기준 분류 체계

| 분류 | 코드 | 의미 | ZeroSite 처리 |
|------|------|------|---------------|
| **즉시 탈락** | R (Reject) | 심사 진행 불가, 자동 탈락 | `status: "REJECT"` |
| **조건부 보류** | H (Hold) | 조건 충족 시 재검토 가능 | `status: "HOLD"` |
| **제외 대상** | E (Exclude) | 법적·정책적 사업 불가 | `status: "EXCLUDE"` |

## 5.3 즉시 탈락 기준 (Rejection Criteria)

### **R-01. 대지면적 500㎡ 미만**

| 항목 | 내용 |
|------|------|
| **근거** | LH 신축매입임대 사업 업무처리지침 제7조 |
| **기준** | 대지면적 < 500㎡ |
| **ZeroSite 진단** | `land_area < 500` → `AI Auto Corrector.validate_land_area()` |
| **조치** | 즉시 탈락 메시지 반환 |

**ZeroSite 구현**:
```python
if land_area < 500:
    return {
        "status": "REJECT",
        "reason": "대지면적 500㎡ 미만 (LH 최소 기준 미달)",
        "recommendation": "인근 필지 통합 검토 (Parcel Cluster Analyzer 실행)"
    }
```

### **R-08. ROI 4% 미만 (사업성 부족)**

| 항목 | 내용 |
|------|------|
| **근거** | LH 내부 매입 기준 (최소 수익률) |
| **기준** | ROI = (LH 매입가 - 총 사업비) / 총 사업비 < 4% |
| **ZeroSite 진단** | `AnalysisEngine.calculate_roi()` |
| **조치** | 즉시 탈락 + 대안 시나리오 제시 |

**ROI 산식**:
```
ROI = (매입가 - (토지비 + 건축비 + 부대비 + 금융비)) / 총사업비 × 100
```
