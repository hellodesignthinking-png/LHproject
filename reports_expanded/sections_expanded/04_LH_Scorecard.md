## 4.1 개요

## 4.3 입지 평가 상세 (85점 만점)

### 4.3.1 역세권 접근성 (25점)

| 거리(m) | 점수 | 비고 |
|---------|------|------|
| 300m 이하 | 25점 | 도보 5분 이내 |
| 301~500m | 20점 | 도보 7분 이내 |
| 501~800m | 15점 | 도보 10분 이내 |
| 801~1,000m | 10점 | 도보 12분 이내 |
| 1,001~1,500m | 5점 | 버스 환승 필요 |
| 1,501m 이상 | 0점 | 역세권 해당 없음 |

**ZeroSite 구현**: `GeoOptimizer.calculate_accessibility_score()` → Gaussian Decay 모델
```
score = 25 * exp(-0.5 * (distance / 300)^2)
```

**가점 조건**:
- GTX·광역철도 1km 이내: +5점
- 2개 이상 노선 환승역 500m 이내: +3점

### 4.3.2 편의시설 접근성 (20점)

| 시설 | 거리 기준 | 배점 | ZeroSite 필드 |
|------|-----------|------|---------------|
| 대형마트·백화점 | 500m 이하 | 5점 | `shopping_score` |
| 은행·우체국 | 300m 이하 | 3점 | `financial_score` |
| 병원(종합·상급종합) | 1km 이하 | 5점 | `hospital_score` |
| 공원·녹지 | 300m 이하 | 4점 | `park_score` |
| 문화·체육시설 | 1km 이하 | 3점 | `culture_score` |

**총점 계산**: 개별 점수 합산 (최대 20점)

**ZeroSite 구현**:
```python
def calculate_convenience_score(poi_list):
    score = 0
    if any(p.category == 'shopping' and p.distance <= 500 for p in poi_list):
        score += 5
    if any(p.category == 'hospital' and p.distance <= 1000 for p in poi_list):
        score += 5
    # ... (생략)
    return min(score, 20)
```

## 4.5 사업성 평가 상세 (80점 만점)

### 4.5.1 ROI 및 수익성 (30점)

| ROI (%) | 점수 | 비고 |
|---------|------|------|
| 10% 이상 | 30점 | 우수 |
| 8~9.99% | 25점 | 양호 |
| 6~7.99% | 20점 | 보통 |
| 4~5.99% | 15점 | 최소 수준 |
| 4% 미만 | 0점 | **부적합** |

**ROI 계산식**:
```
ROI = (LH 매입가 - 총 사업비) / 총 사업비 × 100

총 사업비 = 토지비 + 건축비 + 부대비용 + 금융비용
LH 매입가 = MAX(감정가, 공사비 연동가)
```

**ZeroSite 구현**: `AnalysisEngine.calculate_roi()`

### 4.5.2 매입가 적정성 (25점)

| 매입가/감정가 비율 | 점수 | 비고 |
|--------------------|------|------|
| 105% 이상 | 25점 | 공사비 연동 우수 |
| 100~104.99% | 20점 | 적정 수준 |
| 95~99.99% | 15점 | 보통 |
| 90~94.99% | 10점 | 낮음 |
| 90% 미만 | 0점 | **부적합** |

**공사비 연동제 (2025년)**:
```
인정 공사비 = MIN(실제 공사비, 표준공사비 × 인정비율)
매입가 = 감정가 OR (토지비 + 인정 공사비 + 인정이윤)
```

### 4.5.3 임대료 경쟁력 (15점)

## 4.9 자동 탈락(Rejection) 조건

다음 조건 중 하나라도 해당 시 **자동 탈락** 처리됩니다:

| No. | 탈락 조건 | ZeroSite 검증 |
|-----|-----------|---------------|
| 1 | 대지면적 500㎡ 미만 | `land_area < 500` |
| 2 | 건축 세대수 10세대 미만 | `units < 10` |
| 3 | 건폐율 초과 | `actual_coverage > legal_coverage` |
| 4 | 용적률 초과 | `actual_far > legal_far` |
| 5 | 일조권 미확보 (연속 2시간 미만) | `sunlight_hours < 2` |
| 6 | 맹지 또는 접도 조건 미충족 | `road_width < 4 or road_length == 0` |
| 7 | 녹지지역·개발제한구역 | `zone_type in ['녹지지역', '개발제한구역']` |
| 8 | ROI 4% 미만 | `roi < 4.0` |
| 9 | 토지 소유권 불명확 | `ownership_status != 'confirmed'` |
| 10 | LH 임대료 기준 초과 (시세 80% 이상) | `rent_ratio > 0.8` |

**ZeroSite 구현**:
```python
def check_auto_rejection(analysis_result):
    rejections = []
    if analysis_result['land_area'] < 500:
        rejections.append("대지면적 500㎡ 미만")
    if analysis_result['building_capacity']['units'] < 10:
        rejections.append("건축 세대수 10세대 미만")
    # ... (생략)
    return rejections
```
