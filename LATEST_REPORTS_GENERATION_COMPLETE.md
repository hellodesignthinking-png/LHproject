### 브라우저에서 PDF 변환 (권장)

```

2. Ctrl+P (인쇄)

4. "저장" 클릭

**Firefox:**
```
1. HTML 파일 열기
2. Ctrl+P (인쇄)
3. "Adobe PDF" 또는 "Microsoft Print to PDF" 선택
4. "저장" 클릭

```
1. HTML 파일 열기
2. Cmd+P (인쇄)

4. "저장" 클릭
```
