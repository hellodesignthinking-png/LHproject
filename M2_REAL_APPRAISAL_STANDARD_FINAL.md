## 🔥 핵심 문제와 해결

### 문제 인식
> "방향은 90점 이상 맞는데, '실제 감정평가서' 기준으로 보면 아직 2~3군데가 과잉·오해 소지가 있다."

**지적된 3가지 위험 요소**:
1. ❌ 비중 노출 (50%/30%/20%) → "자동 계산식"처럼 보임
2. ❌ 교과서 톤 ("~하는 방법입니다") → 논문/설명서 느낌
3. ❌ 과도한 면책 문구 ("참고용", "실제 거래가격과 다를 수 있음")

## 🔧 적용된 수정사항

### 1️⃣ 비중 노출 완전 제거

#### Before (문제)
```html
<th>적용 비중</th>
...
<td>50%</td>
<td>30%</td>
<td>20%</td>

<div class="formula-box">
    거래사례 × 50% + 수익환원 × 30% + 공시지가 × 20%
</div>
```

**문제점**:
- 외부 문서에 계산식 노출
- "자동 계산 리포트"처럼 보임
- 실무 감정평가사는 이렇게 쓰지 않음

#### After (해결)
```html
<th>적용</th>
...
<td>중심</td>
<td>참고</td>
<td>참고</td>

<div class="info-box">
    최종 시장가치는 거래사례 분석을 중심으로 판단하였으며,
    수익환원법 및 개별공시지가를 참고하여
    합리적으로 조정·확정하였습니다.
</div>
```

**해결**:
- 비중은 내부 로직으로만 유지
- 외부 문서에는 "중심", "참고", "확정"만 표시
- 감정평가사가 판단한 것처럼 보임

---

### 2️⃣ 교과서 톤 제거

#### Before (문제)
```
거래사례 비교방식
→ "시장에서 형성된 가격 흐름을 바탕으로 대상 토지의 시장가치를 판단하는 방법입니다."

수익환원법
→ "토지에서 발생할 수 있는 예상 순수익을 적정 환원율로 나누어 
    토지의 수익가치를 산정하는 방법입니다."
```

**문제점**:
- "~하는 방법입니다" → 정의 설명
- 논문/교과서 느낌
- 실무 감정평가서 톤이 아님

#### After (해결)
```
거래사례 비교방식
→ "인근 지역의 실제 거래 사례를 수집하여 시점, 위치, 개별 특성을 보정한 후,
    시장에서 형성된 가격 흐름을 바탕으로 대상 토지의 시장가치를 판단하였습니다."

수익환원법 (보조적 검토)
→ "수익환원법은 대상 토지의 이용 가능성과 시장 수익성을 검토하기 위한
    보조적 분석으로 적용하였습니다."
```

**해결**:
- "~습니다" / "~였습니다" → 실무 톤
- 정의 설명 제거
- 전문가가 판단한 문서

---

### 3️⃣ 법적 고지사항 최소화

#### Before (문제)
```
5. 법적 고지사항

평가 기준 및 제한사항:
1. 평가 목적: 본 보고서는 토지의 시장가치 평가를 목적으로...
2. 평가 기준일: {{ appraisal_date }} 기준이며...
3. 데이터 출처: 국토교통부, 한국부동산원...
4. 적용 제한: 본 평가액은 참고용이며, 실제 거래가격과 다를 수 있습니다. ❌
5. 유효기간: 본 보고서는 발행일로부터 6개월간...

주의사항:
- 본 보고서의 내용을 임의로 수정하거나...
- 평가액은 평가기준일 당시의 시장 상황을...
- 개발계획, 인허가 사항 등...
- 본 평가는 건물이 없는 나대지 상태를...
```

**문제점**:
- "참고용", "실제 거래가격과 다를 수 있음" → 책임 회피
- 5개 항목 + 4개 주의사항 → 과도한 면책
- 실무 감정평가서에서는 거의 쓰지 않음

#### After (해결)
```
5. 평가 기준

평가 기준일 및 조건:
- 평가 기준일: {{ appraisal_date }}
- 본 보고서는 평가기준일 현재의 시장 상황을 기준으로 작성되었습니다.

데이터 출처:
- 국토교통부 실거래가 공개시스템
- 한국부동산원 시장 동향 자료
- 국토교통부 개별공시지가
- 지역 시장 조사 자료
```

**해결**:
- 면책 문구 완전 제거
- 2개 섹션만 유지
- 전문 감정평가서 수준
