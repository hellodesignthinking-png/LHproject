
### 6단계: 결과 확인!
자동으로 스크롤되며 결과가 표시됩니다.

## 📊 결과 화면 구성

### 상단 배지
```
✅ 적격  또는  ❌ 부적격
```
한눈에 매입 가능 여부 확인!

### 1️⃣ 기본 정보
```
주소: 서울특별시 강남구 역삼동 679
용도지역: 제2종일반주거지역
건폐율 / 용적률: 60% / 200%
```

### 2️⃣ 건축 규모
```
예상 세대수: 34세대
층수: 4층
주차대수: 17대
```

### 3️⃣ 수요 분석
```
수요 점수: 78/100점
━━━━━━━━━━━━━━━━━━━━━━  78%
```
**핵심 요인:**
- ✓ 1인 가구 비율 31.0%
- ✓ 지하철역 160m (도보 10분 이내)
- ✓ 대학교 115m 거리

### 4️⃣ 리스크 요인
```
⚠️ 리스크 요인
🔴 [유해시설] 주유소 73m 이내 위치
🟡 [유해시설] 공장 144m 이내 위치
```
- 🔴 빨강 = 높은 리스크
- 🟡 노랑 = 중간 리스크
- 🟢 초록 = 낮은 리스크

### 5️⃣ 종합 판단
```
LH 매입 적격: ❌ 부적격
최종 추천: 부적합 - 매입 제외 대상
```

## ❓ 자주 묻는 질문

### Q: 어떤 주소를 입력해야 하나요?
**A:** 도로명 주소 또는 지번 주소 모두 가능합니다.
```
✅ 서울특별시 강남구 역삼동 679
✅ 서울 강남구 테헤란로 123
✅ 경기도 성남시 분당구 정자동 12-3
```

### Q: 면적은 꼭 정확해야 하나요?
**A:** 대략적인 면적으로도 분석 가능합니다. 정확할수록 더 정밀한 결과가 나옵니다.

### Q: 분석 시간이 오래 걸리나요?
**A:** 보통 6-8초 정도 소요됩니다. 카카오맵, 토지규제, 인구통계 등 여러 API를 동시에 조회하기 때문입니다.

### Q: 결과를 저장할 수 있나요?
**A:** 현재는 화면 캡처로 저장하거나, 결과를 복사해서 사용하세요. PDF 저장 기능은 추후 추가 예정입니다.

### Q: 에러가 나면 어떻게 하나요?
**A:** 
- 주소를 다시 확인해보세요
- 예시 버튼으로 테스트해보세요
- 새로고침 후 다시 시도해보세요
