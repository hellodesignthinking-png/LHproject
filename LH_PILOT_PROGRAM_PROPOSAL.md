# ZeroSite LH Pilot Program 제안서
## 신축매입임대 사전심사 지원 시스템 시범 사업

**제출 대상**: 한국토지주택공사(LH) 신축매입임대 사업본부  
**제안 기관**: ZeroSite Development Team  
**제안 일자**: 2025년 12월 14일  
**사업 기간**: 3개월 (2025년 1월 ~ 3월)  
**사업 규모**: 실제 신청 건 20건

---

## 1. 제안 배경

### 1.1 LH 신축매입임대 현황

**시장 규모**:
- 연간 예산: 약 3조원
- 연간 신청 건수: 500~700건
- 평균 승인율: 40~50%
- 재심사율: 약 20%

**주요 과제**:
1. **신청 건수 증가**: 매년 10~15% 증가로 심사 부담 가중
2. **승인율 낮음**: 절반 이상의 신청이 거절되어 불필요한 심사 비용 발생
3. **심사 기준 불명확**: 신청자의 사전 예측 불가로 민원 증가
4. **데이터 불일치**: 감정평가·사업성·정책 판단이 각기 다른 기준 사용

### 1.2 ZeroSite 솔루션 개요

**ZeroSite**는 **감정평가를 기준축으로 하는 통합 분석 플랫폼**으로:

1. **AI 기반 사전심사 예측**: LH 승인 확률을 6가지 요소로 사전 예측
2. **자동화된 보고서 생성**: 30초 이내 5종 보고서 자동 생성
3. **데이터 일관성 보장**: 감정평가를 Single Source of Truth로 고정

**기대 효과**:
- 승인율 향상: 40% → 70%+
- 심사 시간 단축: 평균 2주 → 1주
- 불필요한 신청 감소: 사전 타당성 검토로 부적합 신청 차단

## 2. Pilot Program 개요

### 2.1 사업 목표

**Primary Goal**: ZeroSite의 LH 승인 예측 정확도 검증

**Success Criteria**:
1. **예측 정확도 70% 이상**: ZeroSite 예측 vs 실제 LH 결정 일치도
2. **심사 시간 30% 단축**: Pilot 참여 건의 평균 심사 시간 비교
3. **신청자 만족도 80% 이상**: 사전 예측 서비스에 대한 만족도 조사

### 2.2 사업 개요

**기간**: 3개월 (2025년 1월 1일 ~ 3월 31일)

**규모**: 실제 신청 건 20건
- 서울 지역: 10건
- 경기 지역: 10건

**참여 대상**:
- LH 신축매입임대 신규 신청자 20명
- LH 심사 담당자 3~5명
- ZeroSite 기술팀 2명

**비용**: 무상 (Pilot 기간 동안 ZeroSite 서비스 무료 제공)

### 2.3 진행 방식

#### Phase 1: 사전 준비 (2주)

**Week 1**:
- LH 담당자 대상 ZeroSite 교육 (2시간)
- 신청자 모집 공고 (LH 홈페이지, 이메일)
- 테스트 환경 구축

**Week 2**:
- 신청자 20명 선정 (서울 10, 경기 10)
- ZeroSite 계정 생성 및 사용 교육
- 데이터 수집 절차 확립

#### Phase 2: 본 사업 (10주)

**신청 건별 프로세스**:

```
1. 신청자가 ZeroSite에 토지 정보 입력
   ↓
2. ZeroSite가 30초 이내 분석 완료
   - 감정평가 기반 분석
   - LH AI Judge 예측 (점수/확률/리스크)
   - 시나리오 A/B/C 제시
   ↓
3. 신청자가 ZeroSite 결과 검토
   - 승인 가능성 높음 → LH 신청 진행
   - 승인 가능성 낮음 → 개선 방안 검토 or 신청 보류
   ↓
4. LH 정식 신청 (ZeroSite 보고서 첨부)
   ↓
5. LH 심사팀이 ZeroSite 보고서 참고하여 심사
   ↓
6. LH 최종 결정 (승인/거절)
   ↓
7. ZeroSite 예측 vs 실제 결과 비교
```

**매주 진행 사항**:
- Week 3-12: 신청 건별 분석 및 LH 심사 진행 (주당 2건)
- 매주 금요일: 주간 리뷰 미팅 (30분)

#### Phase 3: 결과 분석 (2주)

**Week 13-14**:
- 20건 전체 결과 분석
- 예측 정확도 계산
- 심사 시간 비교
- 신청자/심사자 만족도 조사
- 최종 보고서 작성

### 2.4 데이터 수집 계획

**수집 데이터** (20건 각각):

| 항목 | 수집 시점 | 데이터 예시 |
|------|----------|------------|
| 토지 정보 | 신청 시 | 주소, 면적, 용도지역, 공시지가 |
| ZeroSite 예측 | 분석 완료 시 | LH Score (0~100), Pass Probability, Risk Level |
| LH 실제 결정 | 심사 완료 시 | 승인/거절, 승인 조건, 거절 사유 |
| 심사 시간 | 심사 완료 시 | 신청일~결정일 (일수) |
| 신청자 만족도 | 사업 종료 시 | 5점 척도 설문 |

**데이터 활용**:
- v42 ML 모델 학습 데이터로 활용 (익명화 처리)
- LH 내부 심사 기준 고도화 참고 자료
- ZeroSite 예측 정확도 개선

## 4. 역할 및 책임

### 4.1 LH 공사

**담당 부서**: 신축매입임대 사업본부

**역할**:
1. **신청자 모집**: Pilot 참여자 20명 모집 지원
2. **심사 진행**: 기존 절차대로 심사 진행 (ZeroSite 보고서 참고)
3. **데이터 제공**: 최종 결정(승인/거절) 및 심사 기간 데이터 제공
4. **피드백 제공**: ZeroSite 예측 vs 실제 판단 차이에 대한 의견

**제공 사항**:
- 신청자 모집 채널 (홈페이지, 이메일)
- LH 심사 담당자 3~5명 배정
- 주간 리뷰 미팅 참석 (매주 금요일 30분)

### 4.2 ZeroSite

**역할**:
1. **시스템 제공**: Pilot 기간 동안 ZeroSite 무료 사용
2. **교육 제공**: LH 담당자 및 신청자 대상 교육
3. **기술 지원**: Pilot 기간 동안 24시간 기술 지원
4. **결과 분석**: 최종 보고서 작성 및 발표

**제공 사항**:
- ZeroSite 플랫폼 접근 권한 (20 accounts)
- 교육 자료 (PPT, 매뉴얼)
- 주간 리뷰 미팅 참석 (기술팀 2명)
- 최종 보고서 (50~100 pages)

### 4.3 신청자

**선정 기준**:
- LH 신축매입임대 신규 신청 예정자
- 서울/경기 지역 토지 보유
- Pilot 참여 동의 (데이터 활용 동의)

**역할**:
1. ZeroSite에 토지 정보 입력
2. ZeroSite 분석 결과 검토
3. LH 정식 신청 시 ZeroSite 보고서 첨부
4. 사업 종료 시 만족도 조사 참여

**혜택**:
- ZeroSite 분석 서비스 무료 (정상가 30만원)
- LH 심사 우선 처리 (Fast Track)
- 사전 타당성 검토로 불필요한 신청 방지

## 6. 예산 및 비용

### 6.1 비용 구조

**ZeroSite 제공 (무상)**:
- 플랫폼 사용료: 30만원/건 × 20건 = 600만원
- 교육 비용: 200만원
- 기술 지원: 300만원
- 최종 보고서: 500만원
- **합계: 1,600만원 (무상 제공)**

**LH 부담 (최소)**:
- 담당자 인건비: 기존 업무 범위 내
- 신청자 모집: 기존 채널 활용
- 주간 미팅: 30분/주 × 10주 = 5시간
- **추가 비용: 거의 없음**

**신청자 혜택**:
- ZeroSite 분석 서비스: 30만원/건 무료
- 20명 × 30만원 = **600만원 절감**

### 6.2 ROI 분석 (LH 관점)

**비용**:
- LH 담당자 시간: 약 50시간 (5시간 × 10주)
- 추정 비용: 약 150만원 (시간당 3만원)

**기대 효과** (연간 기준):
1. **심사 시간 단축 30%**
   - 기존 심사 시간: 500건 × 2주 × 5일 = 5,000인일
   - 절감: 5,000 × 30% = 1,500인일
   - 연간 절감액: 1,500 × 30만원 = **4.5억원**

2. **불필요한 신청 30% 감소**
   - 기존 거절 건: 500건 × 50% = 250건
   - 감소: 250 × 30% = 75건
   - 심사 비용 절감: 75건 × 1주 × 5일 × 30만원 = **1.1억원**

3. **승인율 향상 (40% → 60%)**
   - 추가 승인 건: 500건 × 20% = 100건
   - 추가 공급 세대: 100건 × 50세대 = 5,000세대
   - 공공주택 공급 기여

**연간 ROI**: (4.5억 + 1.1억) / 150만원 = **약 373배**

## 7. 리스크 관리

### 7.1 잠재 리스크

**Risk 1: 신청자 모집 부족**
- 확률: 중간
- 영향: 높음
- 대응: 
  - 사전 홍보 강화 (LH 홈페이지, SNS)
  - 인센티브 제공 (무료 분석, Fast Track)
  - 모집 기간 연장 (2주 → 4주)

**Risk 2: 예측 정확도 낮음**
- 확률: 중간
- 영향: 매우 높음
- 대응:
  - Success Criteria를 70%로 현실적으로 설정
  - 예측 오류 원인 분석 및 개선 계획 수립
  - v42 ML 전환으로 정확도 향상 계획

**Risk 3: 시스템 장애**
- 확률: 낮음
- 영향: 중간
- 대응:
  - 24시간 기술 지원 체계
  - 백업 시스템 구축
  - 장애 발생 시 48시간 이내 복구

**Risk 4: 데이터 보안 이슈**
- 확률: 낮음
- 영향: 매우 높음
- 대응:
  - 개인정보 수집 최소화 (주소만, 소유주 정보 제외)
  - 데이터 암호화 (AES-256)
  - 익명화 처리 후 ML 학습 데이터 활용

### 7.2 Exit Strategy

**조기 종료 조건**:
1. 신청자 모집 50% 미달 (10명 미만)
2. 시스템 장애 누적 7일 이상
3. 예측 정확도 30% 미만 (명백한 실패)

**조기 종료 시 조치**:
- 참여 신청자에게 무료 분석 서비스 제공 (보상)
- LH에 원인 분석 보고서 제출
- 3개월 후 재도전 제안
