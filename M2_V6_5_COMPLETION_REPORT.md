1. **표 중심 레이아웃** - 데이터를 표로 깔끔하게 정리
2. **시각적 강조** - 핵심 숫자들을 크게 표시
3. **섹션별 명확한 구분** - 페이지별 명확한 구조
4. **전문적인 디자인** - 업계 표준에 맞는 디자인

## 🎨 디자인 특징

### 1. 색상 체계
```css
Primary:   LH Blue (#005BAC)
Secondary: Light Blue (#2165D1)
Success:   Green (#28A745)
Warning:   Yellow (#FFC107)
Text:      Dark Gray (#333)
Background: Light Gray (#F5F5F5)
```

### 2. 타이포그래피
- **본문**: Noto Sans KR, 10pt, line-height 1.6
- **제목**: 20pt (페이지 타이틀), 16pt (섹션), 12pt (서브섹션)
- **숫자 강조**: 36pt (대형), 18pt (중형)

### 3. 레이아웃 요소
- **표 (Tables)**: 데이터 정리용, zebra striping, hover effect
- **카드 (Metric Cards)**: 핵심 지표 강조
- **정보 박스 (Info Boxes)**: 설명/계산식 표시
- **그리드 (Grid)**: 2열/3열 레이아웃
