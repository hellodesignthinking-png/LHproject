## 📄 새 보고서 구조

### I. 보고서 개요 (Introduction)
```
본 보고서는 LH(한국토지주택공사) 신축매입임대사업을 위한 
공급 유형 결정을 목적으로 작성되었습니다. 
대상지의 입지적 특성, 주변 생활인프라 현황, 교통 접근성, 
인구 구조 등을 종합적으로 분석하여 최적의 주택 공급 유형을 
도출하고자 합니다.

분석 방법론으로는 POI(Point of Interest) 기반 입지 분석, 
통계청 인구 데이터 분석, 주변 부동산 시장 동향 분석을 
활용하였으며, 각 공급 유형별 적합도를 정량적으로 평가하였습니다.
분석 신뢰도는 85.0%로 평가되었습니다.
```

### II. 대상지 입지 분석 (Location Analysis)

#### 2.1 교통 접근성 분석
```
대상지의 교통 접근성을 평가한 결과, 반경 1km 이내에 
지하철역 2개소, 버스정류장 5개소가 위치하고 있습니다.

특히 복수의 지하철역이 인접해 있어 대중교통 이용이 
매우 편리한 역세권 입지로 평가됩니다.
이는 직장과 주거의 분리가 뚜렷한 청년층 및 신혼부부 세대에게 
높은 선호도를 보일 것으로 판단됩니다.
```

**POI 데이터 기반 자동 분석**:
- 지하철역 ≥ 2개 → "역세권 입지" 서술
- 지하철역 = 1개 → "양호한 접근성" 서술
- 지하철역 = 0개 → "버스 노선 의존" 서술

#### 2.2 생활편의시설 분석
```
일상생활에 필수적인 생활편의시설의 분포 현황을 조사한 결과, 
반경 500m 이내에 편의점 10개소, 병원 2개소가 위치하며, 
반경 1km 이내에 학교 3개소, 공원 4개소가 분포하고 있습니다.

특히 편의점 밀도가 매우 높아 1인 가구 및 맞벌이 가구의 
생활 편의성이 탁월합니다. 이는 청년층 및 신혼부부가 선호하는 
입지 조건으로, 해당 세대를 타겟으로 한 공급 유형이 적합할 것으로 
판단됩니다.
```

**자동 서술 로직**:
- 편의점 ≥ 8개 → "1인 가구 최적화" 서술
- 학교 ≥ 3개 → "교육 여건 양호" 서술
- 공원 ≥ 2개 → "쾌적한 주거환경" 서술

### III. 공급 유형별 적합성 평가 (Type Evaluation)

```
LH 신축매입임대사업의 주요 공급 유형(청년형, 신혼부부형, 
고령자형 등)에 대하여 대상지의 입지 특성, 주변 인구 구조, 
수요 예측 등을 종합적으로 고려하여 각 유형별 적합도를 
평가하였습니다.

평가 항목으로는 교통 접근성, 생활편의시설, 주변 인구 특성, 
시장 수요 등을 포함하였으며, 각 항목별 가중치를 적용하여 
정량적 점수를 산출하였습니다.
```

**유형별 점수 테이블 + 평가 근거**:

| 공급 유형 | 적합도 점수 | 주요 평가 근거 |
|-----------|-------------|----------------|
| 청년형 | 85.3% | 청년층 1인 가구 특성상 역세권 및 편의시설 접근성이 중요하며, 대상지는 이러한 조건을 충족함 |
| 신혼부부형 | 78.2% | 신혼부부 세대는 교육시설 및 공원 접근성을 중시하며, 중소형 평형 선호도가 높음 |
| 고령자형 | 65.8% | 고령자는 병원 접근성 및 조용한 주거환경을 선호하나, 대상지는 상대적으로 활성화된 지역 |

```
위 표에 나타난 바와 같이, 각 공급 유형별 적합도 점수는 
입지 특성, 주변 인프라, 예상 수요층 등을 종합적으로 반영한 결과입니다.
최고 점수를 기록한 유형이 가장 적합한 공급 유형으로 판단되나, 
실제 사업 추진 시에는 LH의 사업 정책, 예산, 지역 수요 동향 등을 
추가로 고려하여 최종 결정할 필요가 있습니다.
```

### IV. 종합 판단 및 권고사항 (Conclusion & Recommendations)

#### 4.1 판단 근거
```
종합 분석 결과, 대상지는 청년형(소형주택) 공급 유형이 
가장 적합한 것으로 평가되었습니다.
이는 다음과 같은 입지적 특성 및 수요 분석 결과에 기반합니다:
```

**✅ 입지 강점 요인** (자동 생성):
- 복수의 지하철역이 인접하여 교통 접근성이 탁월함
- 편의점 등 생활편의시설이 풍부하여 1인 가구 생활에 최적화됨
- 교육시설 접근성이 우수하여 자녀 교육 여건이 양호함
- 공원 및 녹지 공간이 인접하여 쾌적한 주거환경 제공

**⚠️ 입지 약점 요인 및 개선 방안**:
- 주차 공간 확보가 어려움 (파이프라인 데이터 기반)

#### 4.2 사업 추진 시 권장사항

**청년형 공급 시**:
- **평형 구성**: 소형 평형(전용면적 30~45㎡) 위주로 설계하여 청년층의 경제적 부담을 최소화
- **공용시설**: 공유 오피스, 라운지 등 1인 가구를 위한 커뮤니티 시설 배치
- **주차시설**: 청년층의 차량 보유율을 고려하여 법정 주차 대수보다 축소 가능

**신혼부부형 공급 시**:
- **평형 구성**: 중소형 평형(45~60㎡) 중심, 향후 자녀 출산 고려한 확장 가능 구조
- **공용시설**: 육아 지원 시설(놀이방, 유모차 보관소 등) 배치
- **주차시설**: 세대당 1대 이상의 주차 공간 확보 권장

#### 4.3 결론
```
본 분석에서는 대상지의 입지 특성, 주변 인프라, 교통 접근성, 
예상 수요층 등을 종합적으로 검토한 결과, 청년형(소형주택) 공급이 
가장 적합한 것으로 판단하였습니다.

다만, 최종 사업 추진 시에는 LH의 사업 정책 방향, 예산 제약, 
지역 내 경쟁 공급 현황 등을 추가로 고려하여 의사결정을 
진행할 것을 권고합니다.

또한, 본 보고서의 분석 결과는 현재 시점의 데이터 및 
시장 상황을 기반으로 하고 있으므로, 사업 착수 시점에 
변동된 여건이 있을 경우 재분석이 필요할 수 있습니다.
```

## 🎯 자동 서술 생성 로직

### POI 데이터 → 자동 분석 서술

```python
# 교통 접근성
if subway_count >= 2:
    → "복수의 지하철역이 인접해 있어 대중교통 이용이 매우 편리한 역세권 입지"
elif subway_count == 1:
    → "지하철역이 도보 거리 내에 위치하여 출퇴근 접근성이 양호"
else:
    → "버스 노선이 잘 발달되어 있어 기본적인 대중교통 접근성은 확보"

# 생활편의시설
if convenience_count >= 8:
    → "편의점 밀도가 매우 높아 1인 가구 및 맞벌이 가구의 생활 편의성이 탁월"
if school_count >= 3:
    → "교육시설 접근성이 우수하여 자녀를 둔 신혼부부 또는 일반 가구에게 매력적"
if park_count >= 2:
    → "공원 및 녹지 공간이 인근에 다수 분포하여 쾌적한 주거환경 제공"
```

### 유형별 맞춤형 권장사항

```python
if "청년" in selected_type_name:
    → 소형 평형(30~45㎡) + 공유 오피스 + 축소 주차
elif "신혼" in selected_type_name:
    → 중소형(45~60㎡) + 육아 시설 + 세대당 1대 주차
else:
    → 다양한 평형 믹스 + 공동 시설 + 시장 조사 권장
```

| 권장사항 | ❌ 일반적 문구 | ✅ 유형별 맞춤형 상세 가이드 |
