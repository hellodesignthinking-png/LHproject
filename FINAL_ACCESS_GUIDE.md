## 🔧 문제 해결 (Troubleshooting)

### **❓ "이전 버전이 보여요"**

**해결책**:
```
1. 브라우저 캐시 완전 삭제:
   - Chrome: 설정 > 개인정보 및 보안 > 인터넷 사용 기록 삭제
   - 시간 범위: "전체 기간" 선택
   - 삭제 항목: "캐시된 이미지 및 파일" 체크
   
2. 시크릿 모드로 테스트:
   - Chrome: Ctrl + Shift + N
   - Firefox: Ctrl + Shift + P
   
3. 다른 브라우저에서 테스트
```

### **❓ "분석이 실행되지 않아요"**

**체크리스트**:
```
✅ 주소가 올바르게 입력되었나요?
   → "시도 구/군 동 번지" 형식 확인
   
✅ 토지면적이 입력되었나요?
   → 숫자만 입력 (㎡ 단위)
   
✅ 네트워크 연결이 정상인가요?
   → 브라우저 콘솔(F12) 확인
   
✅ API 서버가 정상 작동 중인가요?
   → Health Check URL 접속 확인
```

### **❓ "PDF 다운로드가 안돼요"**

**해결책**:
```
1. 팝업 차단 해제:
   - 주소창 오른쪽 팝업 차단 아이콘 클릭
   - "항상 허용" 선택
   
2. 다운로드 폴더 권한 확인
   
3. 브라우저 재시작 후 재시도
```
