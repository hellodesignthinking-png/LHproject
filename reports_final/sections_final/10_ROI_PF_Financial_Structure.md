## 10.3 IRR (Internal Rate of Return) 분석

### 10.3.1 IRR 계산식

```
IRR = r (단, NPV = 0을 만족하는 할인율 r)

NPV = ∑[CF_t / (1+r)^t] = 0

여기서:
CF_t = t 시점의 현금흐름
r = 내부수익률 (IRR)
t = 시간 (개월)
```

### 10.3.2 IRR 계산 예시 (Python)

```python
import numpy as np

def calculate_irr(cash_flows):
    """
    현금흐름 기반 IRR 자동 계산
    """
    return np.irr(cash_flows) * 12  # 연간 IRR로 변환

# 현금흐름 예시 (18개월 프로젝트)
cash_flows = [
    -30,  # 0개월: 자기자본 투입 -30억원
    0,    # 1개월
    0,    # 2개월
    ...
    0,    # 17개월
    +46   # 18개월: 순이익 +46억원
]

irr = calculate_irr(cash_flows)
print(f"IRR: {irr:.2f}%")  # 예상: 24.5%
```

## 10.4 NPV (Net Present Value) 분석

### 10.4.1 NPV 계산식

```
NPV = ∑[CF_t / (1+WACC)^t] - 초기 투자액

여기서:
WACC = 가중평균자본비용 (Weighted Average Cost of Capital)
WACC = (E/V) × Re + (D/V) × Rd × (1-Tc)

E = 자기자본 (30억원)
D = 타인자본 (70억원, PF+HUG)
V = 총 자본 (100억원)
Re = 자기자본비용 (12% 가정)
Rd = 타인자본비용 (4%)
Tc = 법인세율 (22%)
```

### 10.4.2 WACC 계산

```
WACC = (30/100) × 12% + (70/100) × 4% × (1-0.22)
     = 3.6% + 2.184%
     = 5.784% ≈ 5.8%
```

### 10.4.3 NPV 계산 예시

```python
def calculate_npv(cash_flows, wacc):
    """
    현금흐름 및 WACC 기반 NPV 자동 계산
    """
    npv = 0
    for t, cf in enumerate(cash_flows):
        npv += cf / ((1 + wacc) ** (t / 12))  # 월단위 → 연단위 변환
    return npv

wacc = 0.058
cash_flows = [-30, 0, 0, ..., +46]
npv = calculate_npv(cash_flows, wacc)
print(f"NPV: {npv:.2f}억원")  # 예상: +12.5억원
```
