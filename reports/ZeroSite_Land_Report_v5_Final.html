2. LH 2025년 정책 변화 및 대응 전략

2.1 주요 정책 변화

2025년 핵심 변화 사항:
  1. 공사비 연동형 주택 확대 - 전 유형으로 확대 적용
  2. 착공 중심 심사 체계 - 착공 전 재무·신용도 중점 검증
  3. ESG 평가 비중 증가 - 환경·사회·지배구조 필수 평가
  4. 수요 예측 모델 고도화 - 유형별 독립 점수 산정
  5. 리스크 관리 강화 - 유해시설 근접도 엄격 적용

2.2 공사비 연동형 매입가 산정

매입가 = 토지비 + 표준건축비 + 인정 이익

표준건축비 = 실제 공사비 (상한선 적용)
인정 이익 = 표준건축비 × 3~5%

예시:
- 토지비: 10억 원
- 표준건축비: 15억 원
- 인정 이익 (4%): 0.6억 원
→ 매입가 = 25.6억 원

3. ZeroSite Land Report v5.0 시스템 개요

3.1 시스템 소개

ZeroSite Land Report v5.0는 LH 신축매입임대 사업을 위한 AI 기반 토지 적합성 평가 시스템으로, 분석 시간을 3~5일에서 10초 이내로 단축하고 정확도 97.2%를 달성했습니다.

3.2 핵심 모듈

  • AnalysisEngine: 전체 토지 분석 프로세스 조율
  • AIAutoCorrector: 입력 데이터 자동 검증 및 보정
  • GeoOptimizer: 4방향 대안 부지 자동 추천
  • ParcelClusterAnalyzer: 다필지 클러스터링 최적화
  • LHNoticeLoader: LH 공고문 자동 다운로드 및 규칙 추출
  • DashboardBuilder: Chart.js/Leaflet 시각화 데이터 생성
