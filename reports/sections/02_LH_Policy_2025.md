# 2. LH 2025년 정책 변화 및 대응 전략

## 2.1 개요

2025년 LH(한국토지주택공사) 신축매입임대 사업은 **공사비 연동형 확대**, **착공 중심 심사 체계**, **ESG 평가 강화** 등 주요 정책 변화를 시행하고 있습니다. 본 장에서는 이러한 변화를 상세히 분석하고, ZeroSite Land Report v5.0의 대응 전략을 제시합니다.

---

## 2.2 정책 변화 상세 분석

### 2.2.1 공사비 연동형 주택 확대

**변경 내용**:
- 기존: 일부 유형(청년, 신혼부부)만 공사비 연동형 적용
- 2025년: 전 유형으로 확대 (일반형, 고령자형 포함)

**공사비 연동형 매입가 산정 방식**:
```
매입가 = 토지비 + 표준건축비 + 인정 이익

표준건축비 = 실제 공사비 (상한선 적용)
인정 이익 = 표준건축비 × 3~5%

예시:
- 토지비: 10억 원
- 표준건축비: 15억 원 (연면적 1,000㎡ × 150만원/㎡)
- 인정 이익: 15억 × 4% = 0.6억 원
→ 매입가 = 10억 + 15억 + 0.6억 = 25.6억 원
```

**ZeroSite 대응**:
- `BuildingCalculator` 모듈에 공사비 연동형 계산식 적용
- 지역별 표준건축비 데이터베이스 구축 (2024~2025 LH 공고 기준)
- 수익성 시뮬레이션 기능 추가

---

### 2.2.2 착공 중심 심사 체계 강화

**신규 프로세스**:
```
[1단계] 사업 계획 심사
   ↓
[2단계] 협약 체결 (조건부)
   ↓
[3단계] 착공 승인 심사 ⭐ (강화)
   - 신용도 재검증
   - 자금 조달 확인
   - 설계 적정성 검토
   ↓
[4단계] 공사 진행 및 감리
   ↓
[5단계] 준공 및 매입
```

**착공 심사 강화 항목**:
1. **재무 건전성**
   - 최근 3개년 재무제표
   - 부채비율 200% 이하 권장
   - 유동비율 150% 이상 권장

2. **신용도**
   - 신용등급 BB+ 이상 (건설사)
   - 최근 3년 내 부도·파산 이력 없음

3. **자금 조달 계획**
   - 총 사업비의 30% 이상 자기자본
   - 금융기관 대출 승인서 제출

4. **설계 적정성**
   - 건축 설계도서 완성도 80% 이상
   - 구조 안전 진단서
   - 에너지 성능 평가서

**ZeroSite 대응**:
- `LHCriteriaChecker`에 착공 심사 항목 자동 체크리스트 생성
- 재무 지표 자동 계산 (부채비율, 유동비율)
- 설계 완성도 체크리스트 제공

### 2.2.3 ESG 평가 비중 증가

**ESG 평가 배점 (2025년 신설)**:

| 항목 | 세부 내용 | 배점 |
|------|----------|------|
| **환경(E)** | 제로에너지 건축물 인증 (ZEB 3등급 이상) | 10점 |
| | 신재생에너지 설치 (태양광 등, 총 에너지의 20% 이상) | 5점 |
| | 친환경 자재 사용 (재활용 자재 30% 이상) | 5점 |
| **사회(S)** | 취약계층 우선 공급 비율 (전체의 30% 이상) | 8점 |
| | 지역사회 기여 (일자리 창출, 문화공간 제공) | 7점 |
| **지배구조(G)** | 내부통제 시스템 구축 | 5점 |
| | 투명한 하도급 관리 | 5점 |
| **합계** | | **45점** |

**평가 기준**:
- 30점 이상: 우수 (가점 적용)
- 20~29점: 보통
- 20점 미만: 미흡 (감점 가능)

**ZeroSite 대응**:
- ESG 점수 자동 계산 모듈 (v6.0 예정)
- 제로에너지 건축물 설계 가이드 제공
- 지역사회 기여 방안 템플릿

### 2.2.4 수요 예측 모델 고도화

**기존 vs 신규 비교**:

| 구분 | 기존 (2024) | 신규 (2025) |
|------|------------|------------|
| **평가 항목** | 인구 통계, 교통 접근성 | + 편의시설 접근성, 개발계획 반영 |
| **유형별 분석** | 통합 수요 점수 | 유형별 독립 점수 산정 |
| **가중치** | 고정 (입지 40%, 수요 60%) | 동적 조정 (입지 30~40%, 수요 25~35%) |
| **데이터 소스** | 행정안전부 인구통계 | + 국토교통부 개발계획, 교육청 학군 정보 |

**신규 수요 점수 산식**:
```
수요점수 = Σ(유형별 요소 × 가중치)

청년형:
= (지하철 점수 × 0.25) + (대학 점수 × 0.20) + (청년인구 비율 × 0.25)
  + (상업시설 점수 × 0.15) + (취업 기회 × 0.15)

신혼·신생아형:
= (초등학교 점수 × 0.25) + (어린이집 점수 × 0.20) + (공원 점수 × 0.20)
  + (병원 점수 × 0.15) + (주거환경 × 0.20)

다자녀형:
= (학군 점수 × 0.30) + (공원 점수 × 0.20) + (주차공간 × 0.15)
  + (면적 적정성 × 0.20) + (안전성 × 0.15)

고령자형:
= (병원 점수 × 0.30) + (복지시설 점수 × 0.25) + (저층 선호 × 0.20)
  + (대중교통 × 0.15) + (편의시설 × 0.10)
```

**ZeroSite 대응**:
- `_calculate_type_demand_scores()` 메서드에 신규 산식 적용
- 실시간 인구통계 API 연동 (MOIS API)
- 개발계획 데이터베이스 구축 (국토교통부 API)

### 2.2.5 리스크 관리 강화

**신규 필수 검증 항목**:

1. **유해시설 근접도 (강화)**
   ```
   [LH 매입 제외 기준]
   - 주유소: 25m 이내 → 자동 탈락
   - 위험물 저장소: 50m 이내 → 자동 탈락
   - 장례식장: 100m 이내 → 고위험 (재검토 필요)
   
   [감점 기준]
   - 고압전선: 50m 이내 → -10점
   - 철도·고속도로: 100m 이내 → -5점
   - 공장: 200m 이내 → -8점
   ```

2. **토지이용규제 확인**
   - 개발행위허가 제한 여부
   - 문화재 보호구역 저촉 여부
   - 군사시설 보호구역 저촉 여부

3. **환경 저해 요인**
   - 소음 수준 (환경부 기준 주간 65dB 이하)
   - 대기질 (미세먼지 PM2.5 연평균 35㎍/㎥ 이하)

**ZeroSite 대응**:
- `LandRegulationService`에 전국 유해시설 DB 구축
- 거리 계산 알고리즘 (Haversine Formula 적용)
- 자동 리스크 등급 분류 (Critical / High / Medium / Low)

## 2.3 유형별 정책 변화 상세

### 2.3.1 청년형 (20~39세)

**2025년 변경사항**:
- 면적 기준: 기존 16~36㎡ → 확대 18~40㎡
- 임대 기간: 최대 6년 → 8년으로 연장
- 소득 기준: 도시근로자 월평균 소득 100% 이하 → 120% 이하

**공급 우선 지역**:
- 서울: 강남구, 서초구, 송파구, 마포구
- 경기: 성남시, 고양시, 수원시
- 인천: 연수구, 남동구

**ZeroSite 분석 포인트**:
- 지하철역 500m 이내 필수
- 대학교 2km 이내 가점
- 청년 인구 비율 20% 이상 선호

---

### 2.3.2 신혼·신생아형 I/II

**I형 (혼인 신고 7년 이내)**:
- 면적: 36~50㎡
- 임대 기간: 최대 10년
- 소득 기준: 도시근로자 월평균 소득 100% 이하

**II형 (자녀 2명 이상 또는 신생아 포함)**:
- 면적: 50~85㎡
- 임대 기간: 최대 20년 (자녀 2명 이상 시)
- 소득 기준: 도시근로자 월평균 소득 120% 이하

**2025년 강화사항**:
- 어린이집 접근성 필수 (500m 이내)
- 초등학교 학군 평가 반영
- 공원·놀이터 접근성 중요도 증가

---

### 2.3.3 다자녀형 (자녀 3명 이상)

**2025년 변경사항**:
- 면적 기준: 85~110㎡ (기존 대비 상한 확대)
- 임대 기간: 최대 20년 (자녀 막내 만 20세까지)
- 우선 공급 지역 확대 (경기 외곽, 지방 광역시 포함)

**필수 요건**:
- 주차 공간 세대당 1.5대 이상
- 초등학교 800m 이내 필수
- 공원 500m 이내 권장

---

### 2.3.4 고령자형 (65세 이상)

**2025년 신설 항목**:
- 면적: 36~50㎡ (1~2인 가구 기준)
- 임대 기간: 평생 (단, 소득 재심사 3년마다)
- 계약 조건: 무주택 또는 1주택 보유자 (보유 주택 처분 조건)

**필수 시설 접근성**:
- 병원 300m 이내 필수
- 복지관 500m 이내 권장
- 저층 선호 (3층 이하 또는 엘리베이터 필수)

## 2.4 심사 통과 전략

### 2.4.1 6대 심사 영역 대응

**1. 입지 분석 (배점: 25점)**
- 교통 접근성 (지하철 500m, 버스 200m)
- 편의시설 접근성 (마트, 은행, 병원)
- 개발계획 반영 (신도시, 역세권 개발)

**2. 수요 분석 (배점: 30점)**
- 유형별 독립 수요 점수 (ZeroSite 자동 산정)
- 인구통계 기반 수요 예측
- 경쟁 공급 현황 분석

**3. 사업성 분석 (배점: 20점)**
- 공사비 적정성 (표준건축비 ±10% 이내)
- 수익률 계산 (목표: 5~8%)
- 자금 조달 계획

**4. 법규 검토 (배점: 10점)**
- 용도지역 적합성
- 건축 규제 사전 검토
- 개발행위허가 가능 여부

**5. 기술 심사 (배점: 10점)**
- 구조 안전성
- 에너지 효율 (ZEB 인증)
- 설계 완성도

**6. 재무 심사 (배점: 5점)**
- 신용등급 BB+ 이상
- 부채비율 200% 이하
- 유동성 확보

**합계: 100점** (70점 이상 통과 기준)

### 2.4.2 가점 항목 활용 전략

**가점 최대 15점 추가 가능**:

| 가점 항목 | 배점 | 조건 |
|----------|------|------|
| ESG 우수 | 5점 | ESG 점수 30점 이상 |
| 제로에너지 건축물 | 3점 | ZEB 3등급 이상 인증 |
| 지역 균형 발전 | 3점 | 비수도권 또는 개발 낙후 지역 |
| 혁신 기술 적용 | 2점 | 스마트홈, IoT 등 |
| 신속 착공 | 2점 | 협약 후 3개월 이내 착공 |

**ZeroSite 가점 전략 추천**:
1. ESG 점수 자동 계산 → 목표 30점 달성 방안 제시
2. 제로에너지 건축물 설계 가이드 제공
3. 지역별 가점 기회 분석 (비수도권 우선 검토)
