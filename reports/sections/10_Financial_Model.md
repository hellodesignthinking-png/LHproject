# 10. 재무 및 수익 모델

## 10.1 사업비 구조

```
총 사업비 = 토지비 + 건축비 + 부대비용 + 금융비용

토지비 = 토지면적 × 감정평가액
건축비 = 연면적 × 단위 공사비
부대비용 = 설계비 + 인허가비 + 부담금 (총 사업비의 5~8%)
금융비용 = 총 사업비 × 대출비율 × 금리 × 기간
```

## 10.2 공사비 연동형 매입가

```
매입가 = 토지비 + 표준건축비 + 인정 이익

표준건축비 = 실제 공사비 (상한선 적용)
인정 이익 = 표준건축비 × 3~5%

예시 계산:
- 토지비: 10억 원
- 표준건축비: 15억 원
- 인정 이익 (4%): 0.6억 원
→ 매입가 = 25.6억 원
```

## 10.3 수익률 계산

```
예상 수익률 = (매입가 - 총 사업비) / 총 사업비 × 100

목표 수익률:
- 안정형: 5~8%
- 성장형: 8~12%
```
