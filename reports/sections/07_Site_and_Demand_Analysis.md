# 7. 입지 및 수요 분석

## 7.1 입지 분석 프레임워크

### 3대 핵심 요소
1. **교통 접근성** (가중치 40%)
2. **생활 편의성** (가중치 35%)
3. **환경 쾌적성** (가중치 25%)

## 7.2 수요 점수 산정 공식

```
총 수요점수 = (입지 × 0.30) + (규모 × 0.25) + (사업성 × 0.30) + (법규 × 0.15)
```

### 유형별 특화 분석
- **청년형**: 지하철·대학·상권 중심
- **신혼부부형**: 학교·어린이집·공원 중심
- **다자녀형**: 학군·주차·넓은 면적
- **고령자형**: 병원·복지시설·저층

## 7.3 Multi-Parcel Analysis

### 클러스터링 알고리즘
```
1. 10개 필지 좌표 입력
2. K-means로 3~5개 클러스터 생성
3. 클러스터별 평균 점수 산정
4. 상위 2~3개 클러스터 추천
```
