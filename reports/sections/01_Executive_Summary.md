# 1. Executive Summary (경영진 요약)

## 1.1 보고서 목적

본 보고서는 **ZeroSite Land Report v5.0**의 기술적 구현과 **LH 신축매입임대 사업 운영 매뉴얼 Ver. 2.0**의 정책·전략 내용을 통합하여, AI 기반 토지 적합성 평가 시스템의 전체 운영 프레임워크를 제시합니다.

**핵심 목표**:
- LH 신축매입임대 사업의 2025년 정책 변화에 대응한 시스템 업데이트
- 7개 주거 유형별 독립 수요 점수 산정 모델 구현
- 다중 필지 분석 및 지리적 최적화 기능 제공
- ESG·리스크·사업성을 통합한 종합 분석 체계 확립
- AI 자동화를 통한 심사 대응 효율성 극대화

## 1.3 2025년 LH 정책 변화 핵심 요약

### 주요 변화 사항

1. **공사비 연동형 주택 확대**
   - 기존 유형 외에 공사비 연동형 적용 범위 확대
   - 실제 공사비 기준 매입가 산정 → 사업자 수익성 개선

2. **착공 중심 심사 체계**
   - 심사 → 협약 → 착공 → 준공 순서 강화
   - 착공 전 신용도·재무 상태 중점 검증

3. **ESG 평가 비중 증가**
   - 환경·사회·지배구조 항목 필수 평가
   - 제로에너지 건축물, 친환경 자재 사용 가점

4. **수요 예측 모델 고도화**
   - 인구통계·교통·편의시설 접근성 기반 수요 점수화
   - 지역별 맞춤형 배점 적용

5. **리스크 관리 강화**
   - 유해시설 근접도 엄격 적용 (주유소 25m 이내 탈락)
   - 토지이용규제·개발행위제한 사전 검증

## 1.5 AI 모델 및 알고리즘

### 1.5.1 수요 점수 산정 알고리즘

**공식**:
```
수요점수 = (입지점수 × 0.30) + (규모점수 × 0.25) + (사업성점수 × 0.30) + (법규점수 × 0.15)
```

**유형별 특화 요소**:

| 유형 | 핵심 평가 항목 | 가중치 |
|------|---------------|--------|
| 청년형 | 지하철 접근성, 대학 근접도, 청년 인구 비율 | 지하철 ≤500m: +15점 |
| 신혼·신생아 I/II | 초등학교, 어린이집, 공원 접근성 | 초등학교 ≤400m: +12점 |
| 다자녀형 | 학군, 공원, 주차 공간, 넓은 면적 | 공원 ≤500m: +10점 |
| 고령자형 | 병원, 복지시설, 저층 선호 | 병원 ≤300m: +15점 |

### 1.5.2 지리적 최적화 알고리즘

**4방향 탐색 모델**:
1. 입력 위치에서 반경 1~3km 내 후보지 탐색
2. 북/동/남/서 4방향 각 1~2개씩 선정
3. 접근성·수요·인프라·환경 4개 카테고리 점수화
4. 종합 점수 상위 3~5개 추천

**점수 산정**:
```
최적화점수 = (접근성 × 0.35) + (수요 × 0.30) + (인프라 × 0.20) + (환경 × 0.15)
```

## 1.9 재무 및 수익 모델

### 사업비 구조

```
총 사업비 = 토지비 + 건축비 + 부대비용 + 금융비용

토지비 = 토지면적 × 감정평가액
건축비 = 연면적 × 단위 공사비
부대비용 = 설계비 + 인허가비 + 부담금
금융비용 = 총 사업비 × 대출비율 × 금리 × 기간
```

### 수익 구조 (공사비 연동형 기준)

```
매입가 = 토지비 + 공사비 + 인정이익 (3~5%)

예상 수익률 = (매입가 - 총 사업비) / 총 사업비 × 100

목표 수익률: 5~8% (안정형), 8~12% (성장형)
```
