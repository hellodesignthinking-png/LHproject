### 4. 조정요인 상세 설명 📋
**문제**: 조정 근거가 불명확

**개선 필요 사항**:
```
각 조정요인별 상세 설명 추가:

1. 면적조정 (±5%):
   - 기준: 표준 필지 대비 크기
   - 계산식: (대상 면적 - 표준 면적) / 표준 면적 × 0.05
   - 근거: 감정평가 실무기준 제6조

2. 도로조정 (±10%):
   - 대로 접면: +10%
   - 중로 접면: +5%
   - 소로 접면: 0%
   - 맹지: -15%
   - 근거: 도로 접면에 따른 활용도 차이

3. 형상조정 (±5%):
   - 정형지: +3%
   - 준정형지: 0%
   - 부정형지: -5%
   - 근거: 토지 이용 효율성

4. 경사조정 (±10%):
   - 평지: 0%
   - 완경사: -5%
   - 급경사: -10%
   - 근거: 개발 비용 증가

5. 용도조정 (±5%):
   - 근거: 용도지역별 개발 밀도
   - 제1종 일반주거: 기준
   - 제2종 일반주거: +3%
   - 제3종 일반주거: +5%
   - 준주거지역: +8%

6. 개발조정 (±5%):
   - 재개발/재건축 예정: +5%
   - 도시개발사업 진행: +3%
   - 해당 없음: 0%

7. 시점조정 (월별):
   - 계산식: (기준시점 - 거래시점) × 월별 변동률
   - 근거: 한국감정원 부동산가격동향
```
