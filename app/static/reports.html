모듈을 선택하고 Context ID를 입력하여 보고서를 조회하세요.

사용 방법

  1. Context ID 입력란에 프로젝트의 Context ID를 입력합니다.
  2. 조회할 모듈 버튼(M3/M4/M5/M6)을 클릭합니다.
  3. 하단에 해당 모듈의 보고서가 표시됩니다.
  4. 보고서는 새 탭에서 열어서 인쇄 또는 PDF 저장이 가능합니다.

API 엔드포인트 형식:

/api/v4/reports/{MODULE}/html?context_id={CONTEXT_ID}
