시스템 개요

ZeroSite는 LH 신축매입임대 사업의 타당성 검토를 위한 의사결정 지원 시스템입니다.

본 시스템은 토지정보 분석(M1)부터 최종 종합 판단(M6)까지 6단계 모듈을 통해 실제 데이터 기반의 판단 근거를 제공합니다.

모든 분석 결과는 LH 내부 검토 및 결재 문서로 활용 가능합니다.

M1 토지정보 분석
M3 공급유형 결정
M4 건축 규모 판단
M5 사업성 분석
M6 LH 종합 판단

모듈 상세

모듈 기능 핵심 산출물 상태
M1 토지정보 분석 주소, 대지면적, 용도지역 운영중
M3 공급유형 결정 청년형/신혼/다자녀/고령자 판단 운영중
M4 건축 규모 판단 세대수, 연면적, 주차계획 운영중
M5 사업성 분석 NPV, IRR, ROI, 리스크 분석 운영중
M6 LH 종합 판단 조건부 GO / 재검토 필요 운영중

보고서 조회 엔드포인트

각 모듈의 HTML 보고서는 다음 엔드포인트를 통해 조회할 수 있습니다:

모듈 엔드포인트 설명
M3 /api/v4/reports/M3/html 공급유형 결정 보고서
M4 /api/v4/reports/M4/html 건축 규모 판단 보고서
M5 /api/v4/reports/M5/html 사업성 분석 보고서
M6 /api/v4/reports/M6/html LH 종합 판단 보고서

파라미터: context_id (필수)

예시: /api/v4/reports/M6/html?context_id=1168010100005200012

⚠️ 시스템 핵심 원칙

  • Zero Tolerance: MOC/SAMPLE/기본값 기반 계산은 절대 허용되지 않습니다.
  • Data Integrity: 상위 모듈 데이터가 연결되지 않은 상태에서 분석 결과를 생성하지 않습니다.
  • Conditional Decision: 무조건 GO 또는 A등급 판정은 존재하지 않으며, 모든 판정은 조건부 구조입니다.
  • Single Source of Truth: 모든 수치는 단일 Context ID 기반으로 계산됩니다.
