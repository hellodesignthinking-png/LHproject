## 🎨 Step-by-Step 개선 사항

### Step 1️⃣: 프로젝트 요약 카드

#### Before (문제점)
```
❌ RUN_ID만 표시
❌ 분석 방식 불명확
❌ 신뢰도 정보 없음
❌ 기술 중심 UI
```

#### After (해결)
```
✅ 프로젝트 요약 카드
  ├─ RUN_ID: DIRECT_20260101_9fd236ae
  ├─ 분석 방식: ✏️ 직접 입력 (참고용)
  ├─ 신뢰도: LOW (주황색 뱃지)
  ├─ 주소: 서울특별시 강남구 테헤란로 152
  ├─ 면적: 500㎡
  └─ 분석 시각: 2026-01-01 14:32
```

**시각적 구분**:
- `✏️ 직접 입력 (참고용)` - 주황색 뱃지
- `🔗 API 기반 (실제 데이터)` - 녹색 뱃지 (향후)

---

### Step 2️⃣: 6종 보고서 카드 표준화

#### Before (문제점)
```
❌ 버튼 레이아웃 불규칙
❌ 역할별 권한 불명확
❌ HTML/PDF 구분 애매
```

#### After (해결)
```
✅ 표준화된 보고서 카드

[A] 종합 최종 보고서
설명: 전체 분석 결과를 포함한 마스터 보고서 (관리자 전용)

[📄 HTML 보기] [📥 PDF 다운로드]
```

```
[F] 투자자 프레젠테이션
설명: 외부 공유용 요약 자료 (공유 기능 지원)

[📄 HTML 보기] [📥 PDF 다운로드] [🔗 공유하기]
```

**행동 중심 버튼**:
- `📄 HTML 보기` - 새 탭에서 즉시 열기
- `📥 PDF 다운로드` - 파일 자동 다운로드
- `🔗 공유하기` - 외부 공유 링크 생성

### Step 3️⃣: Direct Input 전용 경고 UX

#### 대시보드 상단 고정 배너
```
⚠️ 이 프로젝트는 '직접 입력 주소'를 기반으로 생성된 참고용 분석입니다

• 외부 행정 API(Kakao, VWorld, Data.go.kr)를 조회하지 않았습니다
• PNU, 좌표, 공시지가 등은 추정값입니다
• 법적·행정적 효력이 없습니다
• 검토/아이디어용으로만 사용하세요
```

**조건부 표시**:
- `DIRECT_` RUN_ID → 경고 표시
- API 기반 RUN_ID → 경고 숨김

---

### Step 4️⃣: 공유 UX 명확화

#### 공유 모달 구조
```
🔗 외부 공유 링크 생성
로그인 없이 누구나 볼 수 있는 링크를 만듭니다

⚠️ 공유 전 확인사항
• 이 링크는 로그인 없이 외부에 전달됩니다
• 읽기 전용 (수정 불가)
• 다운로드 가능
• 만료 시 자동 차단됩니다

유효 기간 선택:
[24시간] [7일 ✓] [30일]

[취소] [링크 생성] [📋 복사]
```

**실수 방지**:
- 공유 전 경고 문구 필수
- 만료 시간 명시
- 링크 복사 원클릭

```json

  "detail": "Authentication required",
  "status_code": 401

#### After (해결)
```
┌─────────────────────────────────────┐
│ ❌ 접근 권한 없음                      │
│                                     │
│ 이 보고서는 현재 권한으로 볼 수 없습니다.│

**에러 상황별 메시지**:
| 상황 | 사용자 메시지 |

| API 실패 | "보고서를 로드할 수 없습니다. 다시 시도해주세요" |

## 🎓 핵심 디자인 원칙

### 1. **첫 인상이 전부다**
- 프로젝트 요약 카드를 최상단 배치
- 3초 안에 모든 핵심 정보 파악

### 2. **행동이 먼저, 데이터는 나중**
- "HTML 보기", "PDF 다운로드", "공유하기"
- 기술 용어 대신 동사 사용

### 3. **실수를 미리 차단한다**
- 권한 없는 버튼 disabled
- 공유 전 경고 모달
- 명확한 에러 메시지

### 4. **신뢰는 투명함에서 나온다**
- Direct Input 경고 배너
- 신뢰도 뱃지 (LOW/HIGH)
- 법적 효력 없음 명시

### 5. **설명은 UI 실패의 증거다**
- 모든 요소가 자체 설명
- tooltip 최소화
- 직관적 아이콘
