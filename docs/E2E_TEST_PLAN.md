# ZeroSite Decision OS - End-to-End 실전 검증 계획

## 📋 테스트 케이스 선정

### Case A: GO (강남구 역삼동)
**지번**: 서울특별시 강남구 역삼동 123-45 (가상)
**특성**:
- 제2종일반주거지역
- 대지면적: 1,500㎡
- 건폐율: 60%, 용적률: 200%
- 도로조건: 양호 (폭 12m 도로 접함)
- 역세권: 강남역 800m
- 규제: 단순
- 주변 시세: 안정적

**예상 판단**: GO
**근거**:
- 양호한 입지 (강남역 근접)
- 규제 단순
- 건축 규모 확보 가능 (약 240세대)
- 청년 매입임대 수요 높음
- 사업성 안정 (안전 마진 12%+)

---

### Case B: CONDITIONAL (마포구 상암동)
**지번**: 서울특별시 마포구 상암동 456-78 (가상)
**특성**:
- 제3종일반주거지역
- 대지면적: 1,200㎡
- 건폐율: 50%, 용적률: 250%
- 도로조건: 보통 (폭 8m 도로 접함)
- 역세권: 월드컵경기장역 1.2km
- 주차 공간 제약
- 건축비 상승 우려

**예상 판단**: CONDITIONAL
**근거**:
- 입지 양호하나 역세권 거리 다소 멀음
- 주차 공간 제약 (법정 기준 +10% 필요)
- 건축비 변동 리스크 (상암 지역 공사비 상승)
- 도로 조건 보통 (8m → 소방차 진입 확인 필요)
- 보완 조건 충족 시 사업 가능

**보완 조건**:
- HIGH: 주차 확보 계획 구체화
- MEDIUM: 건축비 상한 설정
- LOW: 소방 진입로 사전 협의

---

### Case C: NO-GO (서초구 반포동 - 고도제한 지역)
**지번**: 서울특별시 서초구 반포동 789-10 (가상)
**특성**:
- 제2종일반주거지역
- 대지면적: 800㎡ (협소)
- 건폐율: 60%, 용적률: 200%
- **고도제한**: 20m (약 6층 이하)
- 도로조건: 폭 6m (협소)
- 경관지구 중복 규제
- 일조권 사선제한 강함

**예상 판단**: NO-GO
**근거**:
- 고도제한으로 층수 제약 (6층 이하)
- 대지면적 협소 (800㎡ → 약 80세대만 가능)
- 세대수 부족으로 LH 사업 최소 규모 미달
- 도로 6m (소방차 진입 불가 리스크)
- 경관지구 중복 규제로 인허가 복잡
- 일조권 사선제한으로 실제 용적률 확보 어려움

**구조적 한계**:
- 고도제한 → 층수 제약 → 세대수 부족
- LH 최소 사업 규모(약 100세대 이상) 미달
- 인허가 과정 복잡 및 장기화 예상

## 🔄 테스트 실행 절차

### Phase 1: 프로젝트 생성 (각 케이스별)
```
POST /api/projects
{
  "project_name": "Case A: 강남구 역삼동 청년임대",
  "land_address": "서울특별시 강남구 역삼동 123-45"
}
```

### Phase 2: M1 FACT 입력 & FREEZE
- 지번 기반 토지 정보 수집
- 대지면적, 용도지역, 건폐율·용적률 확인
- 도로조건, 규제사항 입력
- 데이터 검증 후 FREEZE

### Phase 3: M2~M7 순차 실행
- M2: 토지 매입 적정성 (공시지가 기준)
- M3: 공급유형 선정 (청년/신혼부부 등)
- M4: 건축 규모 산정 (보수적 설계)
- M5: 사업성·리스크 검증 (안전 마진)
- M7: 커뮤니티 계획 (민원 방어)

### Phase 4: M6 종합 판단
- M1~M5+M7 통합 분석
- GO / CONDITIONAL / NO-GO 판단
- 판단 근거 및 보완 조건 제시

### Phase 5: PDF 생성 및 검증
- 통합 보고서 자동 생성
- 표지, 목차, 본문 확인
- LH 제출 가능 수준 검증
