## ⚠️ 주의사항

### Mock 데이터 사용 시 M1 Lock 불가

API 데이터 수집이 실패하여 Mock 데이터를 사용하는 경우:

```
┌────────────────────────────────────────────────┐
│ ⚠️ Mock 데이터 사용 중                          │
│ ────────────────────────────────────────────── │
│ 현재 일부 데이터가 Mock 데이터입니다.           │
│ M1 Lock은 실제 API 데이터 또는 수동 입력된     │
│ 데이터만 허용합니다.                            │
│                                                 │
│ 다음 중 하나를 선택하세요:                      │
│ • 📄 PDF 업로드: 지적도, 토지이용계획확인서    │
│   등을 업로드하여 자동 추출                     │
│ • ✏️ 수동 입력: 각 필드를 직접 수정하여       │
│   정확한 값 입력                                │
│ • 🔑 API 키 설정: Step 0에서 API 키 입력       │
└────────────────────────────────────────────────┘
```

**🔴 Mock 데이터 상태에서는 M1 Lock이 차단됩니다!**

**해결 방법:**
1. **수동 입력**: 모든 필드를 직접 입력 (거래사례 포함)
2. **PDF 업로드**: 토지대장, 토지이용계획확인서 업로드 → 자동 추출 + 거래사례만 수동 입력
3. **API 키 설정**: Step 0에서 Kakao, VWorld, Data.go.kr API 키 입력
