API Quick Start

간단한 API 호출로 보고서를 생성하세요

1. Health Check

curl https://8000-ia7ssj6hrruzfzb34j25f-c81df28e.sandbox.novita.ai/api/v3/reports/health

2. Generate Pre-Report

curl -X POST https://8000-ia7ssj6hrruzfzb34j25f-c81df28e.sandbox.novita.ai/api/v3/reports/pre-report \
  -H "Content-Type: application/json" \
  -d '{
    "appraisal_context": {
      "calculation": {
        "land_area_sqm": 660.0,
        "final_appraised_total": 4154535000
      },
      "zoning": {
        "confirmed_type": "제2종일반주거지역",
        "far": 250.0
      }
    }
  }'

3. Download Report PDF

curl https://8000-ia7ssj6hrruzfzb34j25f-c81df28e.sandbox.novita.ai/api/v3/reports/{report_id}/pdf \
  -o report.pdf
