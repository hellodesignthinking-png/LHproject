## 📝 **사용 방법**

### **Step 1: 접속**
위의 UI 주소로 접속합니다.

### **Step 2: 정보 입력**

**필수 입력 항목** (⭐ 표시):
1. **대상 토지 주소** ⭐
   - 예: `서울특별시 마포구 월드컵북로 120`
   
2. **대지면적** ⭐
   - 단위: ㎡
   - 예: `30.0`
   
3. **주택 유형** ⭐
   - 선택: 청년 / 신혼부부I / 신혼부부II / 고령자 / 다자녀

**선택 입력 항목**:
4. **용도지역**
   - 자동 탐지 (비어있으면 시스템이 자동으로 찾음)
   
5. **토지 감정가**
   - 예: `5000000000` (50억원)

### **Step 3: 보고서 생성**
- **"📄 Expert Edition v3 보고서 생성"** 버튼 클릭
- 약 30-60초 대기 (전문가급 보고서 생성 중)

### **Step 4: 다운로드**
생성 완료 후 두 가지 형식으로 다운로드 가능:
- **📄 PDF 다운로드**: 인쇄 가능한 전문 보고서
- **🌐 HTML 다운로드**: 웹에서 볼 수 있는 버전

## ⚠️ **주의사항**

1. **첫 요청 시간**: 첫 보고서 생성 시 60초 정도 소요될 수 있습니다 (엔진 워밍업)
2. **이후 요청**: 30-40초 정도 소요
3. **PDF vs HTML**: 
   - PDF는 추가 변환 시간 필요 (약 10-15초)
   - HTML은 즉시 다운로드 가능
4. **샌드박스 환경**: 
   - 일정 시간 활동이 없으면 자동 종료될 수 있음
   - 재시작 필요 시 백엔드부터 재가동

## 🔧 **트러블슈팅**

### **문제: "Connection refused" 오류**
**해결**: 백엔드가 꺼졌을 수 있습니다. 재시작 필요.

### **문제: 보고서 생성이 너무 오래 걸림**
**해결**: 최초 요청은 60초까지 소요 가능. 기다려주세요.

### **문제: PDF 다운로드 실패**
**해결**: HTML 먼저 다운로드 후, 브라우저에서 "인쇄 → PDF 저장" 사용
